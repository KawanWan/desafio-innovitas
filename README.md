### 🛸 C-137 Intelligence System | Rick and Morty CRUD

  Este projeto é uma aplicação Full Stack desenvolvida para o processo seletivo da Innovitas. Consiste em um dashboard interdimensional que consome a API oficial do Rick and Morty, permitindo que usuários autenticados gerenciem sua própria unidade de custódia de personagens.

### 🛠️ Tecnologias Utilizadas

Frontend
React + Vite + TypeScript: Interface performática e tipagem estrita.

Tailwind CSS: Estilização HUD (Heads-Up Display) moderna e responsiva.

React Router Dom: Gerenciamento de rotas e proteção de acesso.

Axios: Consumo de APIs externa e local.

Backend
NestJS + TypeScript: Estrutura modular e escalável.

TypeORM + SQLite: Persistência de dados local com banco relacional.

Passport + JWT: Autenticação segura de usuários.

Bcrypt: Criptografia de senhas para segurança de dados.

### 🚀 Como Rodar o Projeto

### 1. Configurar o Backend

  Bash
  cd backend
  npm install
  npm run start:dev
  O servidor backend estará rodando em http://localhost:3000.

### 2. Configurar o Frontend

  Bash
  cd frontend
  npm install
  npm run dev
  A aplicação estará disponível em http://localhost:5173.

### 📌 Funcionalidades Implementadas (CRUD Completo)

[x] Dashboard (Home): Estatísticas globais do multiverso e acesso rápido aos últimos alvos salvos.

[x] Scanner (Personagens): Listagem com paginação e filtro por nome em tempo real.

[x] Dossiê (Detalhes): Visualização completa de dados externos e opção de salvamento (Create).

[x] Unidade de Custódia (Meus Personagens): Listagem local (Read) e remoção de registros (Delete).

[x] Edição Local: Alteração de nomes, status e adição de notas de campo (Update).

[x] Autenticação: Sistema de Login e Cadastro com persistência via LocalStorage.
